HTMLのおさらい

HTML is 何？

Webページの「構造」を決める、文法のこと

基本


                        <!DOCTYPE html>
                        <html lang="ja">
                            <head>
                                <meta charset="UTF-8">
                                <title>タイトル</title>
                                <-- ここに要素を書く -->
                            </head>
                            <body>
                                <-- ここに要素を書く -->
                            </body>
                        </html>

例えば、 <html> </html> で囲まれた中に、 <body> </body> がある。これが「構造」

これを基本として、後は自由に変更していく

DOCTYPE宣言


                        <!DOCTYPE html>
  • どんなファイルなのかをブラウザに教えるもの
  • 一番先頭に書かなければならない

                        <!-- HTML5 -->
                        <!DOCTYPE html>

                        <!-- HTML 4.01 Transitional -->
                        <!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01 Transitional//EN" "http://www.w3.org/TR/html4/loose.dtd">

                        <!-- XHTML 1.0 Transitional -->
                        <!DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.0 Strict//EN" "http://www.w3.org/TR/xhtml1/DTD/xhtml1-strict.dtd">
  • いろいろある
  • でも、HTML5の書き方以外覚える必要はない

head要素


                        <head>
  • 中に、ページのメタ情報(付加情報)を表す要素が入る
  • 画面に表示しないものを入れる。 <title> 等

body要素


                        <body>
  • 中に、ページのコンテンツが入る
  • 画面に表示するものを入れる。 <p> 等

JavaScriptからHTMLを変更する

Webブラウザには、JavaScriptを使って、好きなタイミングで自由にHTMLを変更することのできる仕組みがある

例: 増やす

ここをクリック！！

                        <div id="js-add-click">ここをクリック！！</div>
                        <div id="js-add"></div>
                    

                        var i = 0;
                        // 「ここをクリック！！」の要素を取得
                        var clickEl = document.getElementById('js-add-click');
                        clickEl.addEventListener('click', function(e) {
                            // spanの要素を作成
                            var newEl = document.createElement('span');
                            newEl.textContent = 'test' + i++;

                            // あらかじめ定義していたdiv要素へ、先程のspan要素を追加
                            var containerEl = document.getElementById('js-add');
                            containerEl.appendChild(newEl)
                        }, false);
  • めっちゃ書くの面倒くさい
  • 覚えにくい
  • モノによっては、ブラウザによって挙動が違う場合がある

これを解決してくれるのが…

jQuery

jQueryとは

  • HTMLをJavaScriptを簡単に触れるようにしてくれるモノ(ライブラリ)
  • 他にも、ブラウザ間でのプログラムの違いをなくしてくれる

さっきのプログラム


                        var i = 0;
                        // 「ここをクリック！！」の要素を取得
                        var clickEl = document.getElementById('js-add-click');
                        clickEl.addEventListener('click', function(e) {
                            // spanの要素を作成
                            var newEl = document.createElement('span');
                            newEl.textContent = 'test' + i++;

                            // あらかじめ定義していたdiv要素へ、先程のspan要素を追加
                            var containerEl = document.getElementById('js-add');
                            containerEl.appendChild(newEl)
                        }, false);

jQueryではこう書ける

ここをクリック！！

                        var i = 0;
                        // 「ここをクリック！！」の要素を取得
                        var $click = $('#js-add-click-jq');
                        $click.on('click', function(e) {
                            // spanの要素を作成し、
                            // あらかじめ定義していたdiv要素へ、先程のspan要素を追加
                            var $new = $('<span>').text('test' + i++).appendTo( $('#js-add-jq') );
                        });
